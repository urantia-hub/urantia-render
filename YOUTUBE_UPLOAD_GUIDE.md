# YouTube Upload Guide — UrantiaHub Channel

Reference for uploading Urantia Paper videos to YouTube.

## Per-Video Details

### Title

Generated by `urantia-render metadata --papers {id}`. Format:

- Papers: `{Normalized Title} — Paper {id} | The Urantia Papers`
- Foreword: `Foreword | The Urantia Papers`

Example: `The Universal Father — Paper 1 | The Urantia Papers`

Em-dashes in paper titles are automatically normalized with surrounding spaces, e.g. Paper 118 renders as `Supreme and Ultimate — Time and Space — Paper 118 | The Urantia Papers`.

### Description

Generated by `urantia-render metadata --papers {id}`. To copy:
```bash
cat output/metadata/{id}.json | python3 -c "import json,sys; print(json.load(sys.stdin)['description'])"
```

Template:
```
{First 300 chars of first paragraph}...

Read along with Paper {id} of The Urantia Papers, narrated with AI voice.
Duration: {X} minutes

Chapters:
0:00 Introduction
{timestamp} {Section Title}
{timestamp} {Section Title}
...

Read the full text at https://urantiahub.com
API & developer tools at https://urantia.dev

#UrantiaPapers #UrantiaBook #Spirituality #AudioBook
```

YouTube auto-creates chapters from the timestamps in the description.

### Thumbnail

Generate per paper:
```bash
urantia-render thumbnail --paper {id}
```
Output: `output/thumbnails/thumbnail-{id}.png` (1920x1080 with large text)

Upload as custom thumbnail in YouTube Studio.

## Batch Upload Workflow

```bash
# 1. Render papers
urantia-render render --papers 0-196

# 2. Generate metadata (descriptions with chapters)
urantia-render metadata --papers 0-196

# 3. Generate thumbnails
for i in $(seq 0 196); do urantia-render thumbnail --paper $i; done

# 4. Upload to R2 (backup)
urantia-render upload --papers 0-196

# 5. Upload to YouTube manually (or via YouTube API later)
#    Use metadata from output/metadata/{id}.json
#    Use thumbnail from output/thumbnails/thumbnail-{id}.png
```
